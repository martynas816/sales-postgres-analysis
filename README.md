# Sales Analysis with PostgreSQL

End-to-end SQL analytics project using PostgreSQL and DBeaver.

## Project Overview
This project demonstrates a simple analytics pipeline:
- Raw CSV ingestion
- Staging table
- Clean analytics table
- Business-driven SQL analysis

## Dataset
- sales_data_sample.csv
- 2,823 rows
- Retail sales data

## Pipeline
1. Schema creation
2. Raw staging table
3. CSV load via COPY
4. Clean transformation table
5. Aggregation queries

## Analysis
- Revenue by year
- Revenue by month
- Revenue by product line
- Revenue by country
- Revenue by deal size

## Tools
- PostgreSQL
- DBeaver
- SQL
